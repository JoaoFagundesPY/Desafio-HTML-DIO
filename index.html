O que é HTML?

HTML é a sigla para HyperText Markup Language, que em português significa Linguagem de Marcação de Hipertexto. Ele é a linguagem padrão utilizada para criar páginas web e é composto por uma série de elementos, conhecidos como tags, que definem a estrutura e o conteúdo de um documento HTML.

Tags

As tags são elementos fundamentais no HTML que definem a estrutura e o conteúdo de uma página web. Cada tag tem um propósito específico e é usada para marcar diferentes partes do conteúdo.

São exemplos de tags:

p

A tag p é usada para definir parágrafos em uma página HTML.

H1, H2 , H3, H4, H5, H6

As tags H1, H2, H3, H4, H5 e H6 são usadas para definir títulos e subtítulos em uma página HTML. O número após o "H" indica a hierarquia do título, sendo H1 o mais importante e H6 o menos importante.

Ol, Ul e Li

As tags Ol e Ul são usadas para criar listas ordenadas e não ordenadas, respectivamente. A tag Ol é usada para listas numeradas, enquanto a tag Ul é usada para listas com marcadores.

A tag Li é usada para definir itens em listas HTML, tanto ordenadas quanto não ordenadas.

Strong

A tag Strong é usada para definir texto com forte ênfase, indicando que o conteúdo é importante.

a

A tag a é usada para criar links em uma página HTML.

hr

A tag hr é usada para criar uma linha horizontal em uma página HTML.
